# Artale-Boss-Chaser
一個簡單好上手的 Boss 擊殺紀錄工具，專為玩家設計，用來快速記錄擊殺、追蹤出貨狀態，並預估下一次復活時間。

**這份使用說明僅面向玩家（User）**：以下內容說明網站的功能與操作流程，未包含開發或內部檔案細節。

**主要功能（玩家視角）**

- 快速記錄：選擇左側 Boss 卡片，點按右側的「確認新增紀錄」即可記錄一次擊殺。
- Chain Hunting（連續狩獵）支援：系統在你按下送出時會自動使用當下時間作為擊殺時間，並在新增後自動為下一次準備（頻道自動 +1、掉寶重設為「否」、備註清空），加快高頻操作流程。
- 復活預估：每筆紀錄會顯示該 Boss 的最早 / 最晚復活時間，卡片上會顯示倒數與進度提示，方便判斷是否該派人守點。
- 狀態提示與篩選：卡片會以顏色與文字顯示當前狀態（已存活 / 即將重生 / 冷卻中），並支援篩選查看特定狀態的 Boss。
- 歷史紀錄與管理：下方列表列出近期擊殺紀錄，包含擊殺時間、頻道、是否掉寶、備註與預估復活區間；支援刪除單筆或清除全部紀錄。
- 私密儲存：你的紀錄只儲存在你的瀏覽器，不會自動上傳到任何伺服器（請自行備份以防資料遺失）。

**如何使用（快速步驟）**

1. 在瀏覽器中開啟 `docs/index.html`（或使用任一靜態伺服器提供該檔案）。
2. 點選左側你要紀錄的 Boss 卡片（卡片會顯示該 Boss 名稱與預估復活視窗）。
3. 若需要，先調整頻道與備註；按下右側的「確認新增紀錄」後系統會以當下時間記錄擊殺，並自動為下一筆準備（頻道 +1、掉寶清為否、備註清空）。
4. 在下方的歷史紀錄表格確認或刪除紀錄。

**備份與還原（玩家快速操作）**

- 匯出備份（在瀏覽器 DevTools Console 執行）：

   ```js
   const data = localStorage.getItem('bossKillHistory');
   console.log(data);
   // 複製輸出字串並存成檔案作為備份
   ```

- 匯入還原（在 Console 執行，將 `<PASTE_JSON_STRING>` 換成你的備份字串）：

   ```js
   localStorage.setItem('bossKillHistory', '<PASTE_JSON_STRING>');
   location.reload();
   ```

**常見問題（FAQ）**

- 我可以在手機上使用嗎？

   是的，介面為響應式，可在多數現代瀏覽器的手機或平板上使用。

- 我的資料會上傳到網路嗎？

   不會。資料儲存在你的瀏覽器 LocalStorage。如需跨裝置使用，請手動匯出再匯入。

- 我誤刪了紀錄怎麼辦？

   如果你有先前匯出的備份，可透過匯入還原。否則 LocalStorage 被清除的紀錄將無法復原。
